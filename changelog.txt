Version 1.7 – 2026-03-03
* Strukturendringer
   - Flyttet admin-moduler til egen /admin mappe
   - Skilt autentisering ut i includes/auth.php
   - Innført includes/version.php for sentral versjonshåndtering
   - Separert kartvisning i egen map.php
   - Egen debug-modul (admin/debug.php)
* Autentisering
   - Hybrid autentisering:
   - API Key (X-GPS-API-Key)
   - HTTP Basic Auth (OwnTracks)
   - Rate limiting pr autentiseringstype
* Posisjonshåndtering
   - Deduplisering ved stillstand
   - Accuracy-filter
   - Jitter-filtrering
   - Lagring av fart og høyde
* Statistikk
   - Maks fart
   - Maks høyde
   - Distanseberegning (Haversine)
   - Periodevalg
* Admin UI
   - Fanebasert struktur
   - Separat kartvisning
   - Forbedret loggvisning
   - Egen debug-side
* Teknisk
   - Bedre separasjon av ansvar
   - Mindre coupling mellom UI og backend
   - Klargjort for videre utvidelser

Version 1.6 – 2026-01-02

* Større arkitektur- og funksjonsoppdatering
* Full refaktorering av admin-UI til fanebasert struktur
   - Posisjoner
   - Logger
   - Innstillinger
  - Statistikk
* Flyttet databasekode fra db.php til database.php
* Implementert hybrid autentisering på /location:
   - API-nøkkel (header: X-GPS-API-Key)
   - OwnTracks HTTP Basic Auth
* Lagt til rate limiting per autentiseringstype
   - Egen policy for OwnTracks (burst-vennlig)
   - Egen policy for API-klienter
* Forbedret filtrering og deduplisering av OwnTracks-data:
   - Ignorerer jitter ved stillstand
   - Filtrerer bort dårlig GPS-nøyaktighet
   - Hindrer for hyppige like posisjoner
* Støtte for fart (km/t) lagret i database
* Kart-popup viser nå strukturert informasjon:
   - Klokkeslett
   - Dato
   - Høyde
   - Fart
* Grunnlag lagt for statistikk:
   - Maks fart
   - Maks høyde
   - Distanseberegning
   - Periodevalg (videreutvikles)
* Forbedret debug-logging og feilsøking
* Generell kodeopprydding og bedre mappestruktur
